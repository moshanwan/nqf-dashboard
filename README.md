# نظام متابعة التسكين NQF والاعتماد الأردني

## طريقة التشغيل
افتح ملف `index.html` مباشرة، أو ارفعه على GitHub Pages.

## كيف أعبئ البيانات؟
1. افتح الصفحة.
2. اضغط زر **تحديث** أمام أي برنامج.
3. عدّل:
   - حالة NQF
   - نتيجة تدقيق NQF
   - تواريخ NQF
   - حالة الاعتماد
   - نتيجة تدقيق الاعتماد
   - المسؤول
   - الأولوية
   - الملاحظات
4. اضغط **حفظ** داخل النافذة.
5. اضغط زر **حفظ التعديلات** أعلى الصفحة.

## كيف أعرف البرامج التي لم تعتمد؟
من قائمة التقرير اختر:
- البرامج غير المعتمدة

## كيف أعرف البرامج التي تم تدقيقها وعادت؟
اختر:
- الاعتماد عاد للتعديل
أو:
- التسكين عاد للتعديل

## كيف أحدث البيانات؟
هناك طريقتان:

### الطريقة السهلة
استخدم النظام من المتصفح، ثم اضغط:
- تصدير JSON

وفي أي جهاز آخر:
- استيراد JSON

### طريقة GitHub الرسمية
بعد تصدير JSON، يمكن استخدامه كنسخة احتياطية. أما البيانات الأصلية داخل GitHub فهي في ملف:
- `data.js`

يمكنك تعديل الحالات مباشرة من داخل الصفحة، لكن حفظها يكون على المتصفح الحالي. لجعلها ثابتة للجميع، ارفع نسخة محدثة من `data.js` أو اعتمد ملف JSON كنسخة تشغيلية.

## ملاحظة مهمة
هذا الإصدار يعمل بدون سيرفر. لذلك الحفظ يكون محلياً داخل المتصفح باستخدام LocalStorage. إذا أردت نظام مؤسسي متعدد المستخدمين، الأفضل لاحقاً ربطه بقاعدة بيانات مثل Supabase أو Firebase.
